## Непреложные правила (из AGENTS.md)
1. **Язык общения — только русский.** Отвечай пользователю по-русски.
2. **Язык сайта — английский.** Все имена персонажей и атмосферные описания в карточках — на английском.
3. **Всегда коммить и пушь работу.** После каждого логически завершённого шага: stage изменений → commit с понятным сообщением → `git push origin main`. Сайт деплоится с `main` на GitHub Pages автоматически.

## Что это за проект
Статичный витринный/портфолио-лендинг **Dark Saga Art** — бренд авторских коллекционных арт-кукол и керамических ти-петов (tea pets). Это **витрина для перехода на Etsy, не интернет-магазин** (продажи живут на Etsy).

Стек: чистый **HTML + CSS + vanilla JS**, без фреймворков и сборки. Весь сайт — фактически один файл [index.html](index.html) (HTML, стили и скрипты внутри). Дополнительно [mobile-preview.html](mobile-preview.html) — утилита для просмотра в рамке 390px.

## Как запустить / посмотреть локально
Сборки нет. Достаточно открыть `index.html` в браузере. Для проверки относительных путей к ассетам удобнее поднять локальный сервер:
```bash
python -m http.server 8000
# затем открыть http://localhost:8000/index.html
```
Мобильную верстку смотреть через `mobile-preview.html`.

## Структура контента
- Hero → Tea Pets → Art Dolls (три серии: *Bastards of the Fall*, *Urban Misfits*, *Spores*) → «How to Buy»/соцсети → About → Footer.
- Ассеты по папкам в `assets/` (`tea-pets/`, `bastards/`, `urban/`, `spores/`, `logos/`). Соглашения об именовании — в @AGENTS.md.
- Изображения карточек и логотипы держать квадратными (1:1).

## Чего НЕ делать без запроса
- Не добавлять JS-фреймворки или системы сборки — только vanilla.
- Не менять деление категорий (Tea Pets / Bastards / Urban / Spores).
- Не добавлять e-commerce: цены, размеры, материалы, доставку, наличие — это всё на Etsy, не здесь.
- Не добавлять третий уровень детализации в карточках (попапы/аккордеоны) без явного запроса.

## Работа с изображениями Etsy
- Etsy блокирует прямые GET-запросы к страницам листингов (403). Картинки скачиваются напрямую с CDN `i.etsystatic.com` (URL вида `https://i.etsystatic.com/.../il_fullxfull.XXXX.jpg`).
- Одноразовые скрипты загрузки складывать в `scratch/` (не коммитить в основную историю без нужды).
- Имя персонажа в листинге часто зашито в тексте описания, а не в заголовке.

## Окружение (Windows / PowerShell — из README)
- `gh` после установки через winget может не подхватиться в активной сессии — обновить PATH или перезапустить шелл.
- В PowerShell не использовать `-f` у `gh api` (конфликт с оператором форматирования) — использовать `--field` / `--raw-field`.
- Для неинтерактивного ssh добавлять `-o StrictHostKeyChecking=accept-new`.
