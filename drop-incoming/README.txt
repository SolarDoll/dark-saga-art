КАК СДАВАТЬ НОВЫЙ БАТЧ (без правки кода)
=========================================

На каждого нового чувака — отдельная папка внутри drop-incoming/.
Имя папки латиницей, без пробелов (iron-harlequin, marl, pip).

Внутри папки:
  • фото работы (сколько есть). Лучшее/главное — назови 01.jpg
    или положи первым; квадрат не обязателен, я сам вкадрирую.
  • info.txt — заполни по шаблону:
      – для КУКОЛ (Bastards / Urban Misfits / Spores)  →  _TEMPLATE-doll.txt
      – для ТЕЙ-ПЕТОВ (Tea Spirits)                    →  _TEMPLATE-teapet.txt
    Скопируй нужный шаблон в папку чувака, переименуй в info.txt и заполни.

Пример заполнения (кукла) — в папке EXAMPLE-iron-harlequin/.

ОПИСАНИЕ ПИШЕШЬ СРАЗУ в info.txt:
  • Куклы: mood (фраза на карточке) + legend (история) + profile (Origin/Likes/…).
  • Тей-петы: mood + atmosphere + personality + story.
  • vibe — больше не используется, не заполняй.

ОГРАНИЧЕНИЕ по mood: ≤ ~45 знаков, одна короткая фраза
  (это подпись под именем на карточке; длиннее — карточки разъезжаются).
  legend/story/profile — без ограничений, пиши сколько нужно.

ССЫЛКИ НА ETSY: давай через «Share → Copy link» (длинную, где видно
  /listing/цифры/…). Формат не важен — я сам приведу к нужному виду.
  По возможности НЕ короткую etsy.me — её сложнее раскрыть.
  Если работа ещё не выложена — оставь etsy пустым.

Когда всё сложишь — напиши «батч готов», и я:
  – пережму фото в webp, вкадрирую в квадрат, разложу в assets/works/<серия>/<чувак>/
  – добавлю работы в их разделы-галереи (карточка + деталь Legend/Profile)
  – соберу секцию New Drop в нужной сортировке
    (доступные-новые → доступные-старые → недоступные-новые → недоступные-старые)
  – поставлю статусы Available/Adopted и ссылки Etsy
  – какую работу показать крупно в карусели во время дропа — скажешь (или выберу сам)

Папку drop-incoming/ на боевой сайт не публикуем — это рабочая передача файлов.
